车联网安全专用名词

CSMS Cyber Security Management System 信息安全管理体系认证
VSOC Vehicle Security Operation Center 安全运营中心
VTA Vehibcle Type Approval 车辆型式认证
SOR Specification of Requirement 需求说明
ECU Electronic control Unit 电子控制单元
FOTA Firmware updates Over The Air 固件更新
MOST Media Oriented System Transport 媒体导向系统传输
CAN Control Area Network 控制局域网
LIN Local Interconnect Network 局域互连网络
CSMA/CD Carrier Sense Multiple Access/ Collision Detection 载波侦听多址/冲突检测
TDMA Time Division Multiple Access 时分多址接入
OBD OnBoard Diagnostics 车载诊断
RSU Road Side Unit 路边侧单元
VANET Vehicle Addhoc Networks 车内自组织网络
OBU OnBoard Unit 车载设备
PBL Primary Boot Loader 主引导加载程序
SBL Secondary Boot Loader 辅助引导加载程序
AUTOSAR Automotive Open System Architecture 汽车开放系统体系结构
OEM Original Equipment Manufacturer 原始设备制造商
PCM Powertrain control Module 动力总成控制模块
TCM Transmission Control Module 变速器控制模块
BCM Body Control Module 车身控制模块
ABS Antilock Braking System 防抱死制动系统

ESC Electronic Stability Control 电子车身稳定系统
HVAC Heating Ventilating and Cooling 采暖通风与制冷
TCU Telematics Control Unit 远程通信控制单元
EBCM Electronic Brake Control Module 电子制动控制模块
TPM Trusted Platform Module 可信平台模块
FPGA Field Programmable Gate Array 现场可编程逻辑器件
SoC System on Chip 芯片系统
Inter vehicle communication 车内网
In-vehicle communication 车联网

DMZ Demilitarized Zone 隔离区 内外网防火墙之间的缓冲区，放置一些公共服务器，如论坛
GDPR General Data Protection Regulation 欧盟的条例《通用数据保护条例》
TARA Threat Analysis and Risk Assessment 威胁分析和风险评估。在概念阶段应用的分析技术，以帮助识别功能的潜在威胁，并评估与已识别威胁相关的风险。 识别潜在威胁并评估与这些威胁相关的风险，允许组织优先考虑与威胁相关的后续网络安全活动，因此努力和资源可以集中在最高优先级的威胁。
SSID：Service Set Identifier，服务集标识。SSID技术可以将一个无线局域网分为几个需要不同身份验证的子网络，每一个子网络都需要独立的身份验证，只有通过身份验证的用户才可以进入相应的子网络，防止未被授权的用户进入本网络。
PCB：Printed Circuit Board，中文名称为印制电路板，又称印刷线路板，是重要的电子部件，是电子元器件的支撑体，是电子元器件电气相互连接的载体。由于它是采用电子印刷术制作的，故被称为“印刷”电路板。
