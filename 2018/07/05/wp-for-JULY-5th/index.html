第一关.homework.exe

一，观察程序

任意输入，测试程序

发现可以重复输入，若输入错误，程序会返回固定语句

二，打开OD，观察汇编代码

进入主函数，发现[local.38]表示输入的字符串

将s压入栈，call strlen函数，运行后发现eax中存储strlen(s)的值

将eax中的值赋给[local.4],并与0x1B（转换为十进制为27）相比较

即将字符串s的长度与26相比较，

若不相等，跳转至输出Retry~~的指令处；

若相等，跳转至逐个判断单个字符的指令处

其中发现

00401446|. 31D0  ||xor eax,edx ;

xor指令表示进行亦或运算，两者二进制位上的数，同为0，异为1，储存在前一个数中

可发现[local.13]是与输入字符串相比较的字符串，在内存中找到字符串

Tnfb}eu'`fgn{b|d0xa3gz6ryjc，

再次调试，找到解密后的字符串

Today`s homework is so easy

运行函数，输入正确

第二关.Easy_CrackMe.exe

一.观察程序

输入错误，显示incorrect password

二.用OD调试

1.进入主函数，找到输出incorrect password的语句

2.一步步反推观察cmp指令

3.发现[esp+0X5]与0x61(ACSII:a)比较若！=0才回跳转，说明[esp+0X5]=a是password的一部分

4.若不跳转，那么继续执行语句，发现指令在[esp+0XA]处开辟了两个空间并让‘5y’入栈，经过上一条指令，可以确定[esp]系列指向的是password的地址

5.同样找到输出congeatulation部分的语句，步步反推，观察cmp指令，

6.发现当[esp+0X4]！=0X45(ASCII:E)时跳转，说明[esp+0X4]=E是password的一部分

7.发现逐个字符比较的函数，不难发现入栈的字符串为R3versing,而函数中也是与这个字符串比较，说明是password的一部分，在栈中的开始位置是esp+0X10

8.通过上述过程按顺序得到字符串Ea5yR3versing,运行程序，返回正确。
